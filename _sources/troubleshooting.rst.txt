Troubleshooting
===============
This section describes some common issues that can arise and possible solutions.

Issue #1
--------
Password for postgres was not working...you get locked out
Researched a command that allowed me to reset and change my postgres password and granted me access.

Issue #2
--------
Was encountering several errors due to not activating my virtual environment
Followed steps to activate my virtual environment

Issue #3
--------
You may switch computers, pull code from the repo and try to run app.py. Your virtual environment doesn't follow you
Install, build, activate your virtual environment.
